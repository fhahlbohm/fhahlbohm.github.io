Research

My work is focused on advancing fast and accurate representations for reconstruction and view synthesis from images.

project page / paper / code

A research-friendly Gaussian Splatting framework that integrates and extends existing ideas for significantly improved efficiency.

NVGS: Neural Visibility for Occlusion Culling in 3D Gaussian Splatting

arXiv, 2025
project page / paper / code

A method that combines neural visibility predictions for backface culling with instanced rendering of 3D Gaussian splats.

project page / paper / code

Faster INPC training and rendering with less VRAM and better quality.
